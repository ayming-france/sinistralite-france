```css
.map-section {

  margin-bottom: 24px;

  text-transform: uppercase;
}

.map-wrap {
  overflow: hidden; /* critique: empêche le scroll horizontal à 375px */
  border-radius: 8px;
}

.map-svg {
  display: block;
  width: 100%;
  height: auto;
}

/* Remplissage explicite pour éviter l'héritage de currentColor comme fill */
.map-caisse {
  fill: var(--border-light);
  stroke: var(--bg-elevated);
  stroke-width: 1.2;
  stroke-linejoin: round;
}

/* Forcer l'héritage du fill sur les paths enfants */
.map-caisse path {
  fill: inherit;
}

@media (max-width: 768px) {
  .map-section {
    margin-top: 24px;
  }
}
```

Points critiques:
- `fill: var(--border-light)` sur `.map-caisse` est OBLIGATOIRE — sans lui, les paths héritent de `color` via `currentColor` et apparaissent noirs (cf. pitfall 1 dans RESEARCH.md)
- `overflow: hidden` sur `.map-wrap` est OBLIGATOIRE pour MAP-11 (pas de scroll horizontal à 375px)
- Ne jamais mettre `min-width` sur `.map-svg` ou `.map-wrap`
- Ne pas scoper les règles sous `.nav-icon-wrap` ou `.comp-toggle` (ces sélecteurs gèrent d'autres SVGs)
