  <name>Task 2: Implement tooltip, year change, and ranking logic in js/map.js</name>
  <files>js/map.js</files>
  <action>
Add to js/map.js the following functions and event wiring.

**Module-scoped state:**
```js
const sortState = { at: 'desc', trajet: 'desc' };
```

**Tooltip functions:**

`setupTooltip(svgId, viewType)`:
- Get SVG element by svgId
- Get the shared tooltip element: `document.getElementById('mapTooltip')`
- Add `mousemove` event listener on the SVG element (event delegation):
  - `const g = e.target.closest('[data-caisse]');`
  - If no g: hide tooltip, return
  - Get caisse ID from g.dataset.caisse
  - Find the caisse data from regionalData.caisses by matching ID
  - Get the current year from the year selector: `document.getElementById(viewType + '-yearSelect').value`
  - Get metric value: `caisse[viewType][year]`
  - Set tooltip innerHTML: `<span class="tooltip-name">{caisse.name}</span><span class="tooltip-val">{metric label}: {value.toLocaleString('fr-FR')} ({year})</span>`
  - Where metric label is `viewType === 'at' ? 'Accidents du travail' : 'Accidents de trajet'`
  - Show tooltip: `tooltip.style.display = 'block'`
  - Position with viewport clamping:
    ```js
    let left = e.clientX + 12;
    let top = e.clientY + 12;
    const tw = tooltip.offsetWidth;
    const th = tooltip.offsetHeight;
    if (left + tw > window.innerWidth - 8) left = e.clientX - tw - 12;
    if (top + th > window.innerHeight - 8) top = e.clientY - th - 12;
    tooltip.style.left = left + 'px';
    tooltip.style.top = top + 'px';
    ```
- Add `mouseleave` on SVG: hide tooltip
- Suppress tooltip on touch devices: wrap the above in `if (navigator.maxTouchPoints === 0)` check. On touch devices, tooltip is deferred to Phase 9 mobile panel.

**Ranking function:**

`renderRanking(viewType, year)`:
- Get the ranking list element: `document.getElementById(viewType + '-rankingList')`
- If not found, return
- Build metroVals from regionalData (same as colorierCarte: filter to METRO_IDS, extract val)
- Sort by val according to `sortState[viewType]` (desc or asc)
- Set innerHTML: ordered list items with:
  ```js
  `<li class="ranking-item"><span class="ranking-pos">${i+1}</span><span class="ranking-name">${c.name}</span><span class="ranking-val">${c.val.toLocaleString('fr-FR')}</span></li>`
  ```

**Sort button handler:**

`setupSortButton(viewType)`:
- Get button: `document.getElementById(viewType + '-sortBtn')`
- On click: toggle `sortState[viewType]` between 'desc' and 'asc'
- Update button text: `\u25BC` for desc, `\u25B2` for asc
- Get current year from selector
- Call `renderRanking(viewType, year)`

**Year change handler:**

`setupYearSelector(viewType)`:
- Get select element: `document.getElementById(viewType + '-yearSelect')`
- On change: call `updateMap(viewType, e.target.value)`

**Central update function:**

`updateMap(viewType, year)`:
- Call `colorierCarte(viewType, year, regionalData)` (this already calls renderLegende internally from Plan 01)
- Call `renderRanking(viewType, year)`

**DOMContentLoaded update:**

After the existing `colorierCarte()` calls from Plan 01, add:
