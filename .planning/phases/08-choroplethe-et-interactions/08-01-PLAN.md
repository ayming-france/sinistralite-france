  <action>
Rewrite js/map.js to implement the full choropleth coloring logic. Keep CAISSE_IDS and verifierStructureSVG() from Phase 7 stub.

Add at module scope:
- `const METRO_IDS = [... same 16 as CAISSE_IDS ...]` (reuse existing CAISSE_IDS)
- `let regionalData = null;` module-scoped cache
- `const MIN_COLOR = '#deebf7'; const MAX_COLOR = '#08519c';` (ColorBrewer sequential blue)
- `const DEFAULT_YEAR = '2023';`

Add `interpolateColor(minColor, maxColor, t)` function:
- Parse hex to RGB channels
- Linear lerp each channel by t (clamped 0-1)
- Return `rgb(r,g,b)` string

Fill the `colorierCarte(viewType, year, data)` export body:
- Determine svgId: `viewType === 'at' ? 'france-map-at' : 'france-map-trajet'`
- Get svg element by ID. If not found or no data, return early
- Build metroVals array: filter data.caisses to METRO_IDS, extract `c[metric][year]` where metric is 'at' or 'trajet' (matching viewType)
- Compute min/max over metroVals
- For each metro caisse, compute t = (val - min) / (max - min), set `el.style.fill = interpolateColor(MIN_COLOR, MAX_COLOR, t)`
- If a caisse has no SVG element (DOM-TOM), skip silently (no console.warn)
- If min === max, set t = 0 (all same color)
- Also call `renderLegende()` from inside colorierCarte, passing the legend element ID, min, max, MIN_COLOR, MAX_COLOR

Add `renderLegende(legendElId, minVal, maxVal, minColor, maxColor)` function:
- Get element by legendElId. If not found, return
- Generate 5 swatches (t = 0, 0.25, 0.5, 0.75, 1.0)
- For each swatch, compute color and value label (formatted with toLocaleString('fr-FR'))
- Set innerHTML: `<span class="legend-min">{min}</span><div class="legend-swatches">{swatches}</div><span class="legend-max">{max}</span>`
- Each swatch: `<span class="legend-swatch" style="background:{color}" title="{val}"></span>`

Add async `loadRegionalData()`:
- `const res = await fetch('data/regional-data.json');`
- `regionalData = await res.json();`

Update DOMContentLoaded handler:
- Keep verifierStructureSVG() call
- After that, `await loadRegionalData();`
- Call `colorierCarte('at', DEFAULT_YEAR, regionalData);`
- Call `colorierCarte('trajet', DEFAULT_YEAR, regionalData);`

MAP-07 integration: The existing page nav already shows #view-at or #view-trajet. Each view has its own SVG and map section. colorierCarte is called once per view at load time. No separate AT/Trajet toggle inside the map is needed. The nav switching already shows the correct map section.

Important: keep `export` on colorierCarte. The DOMContentLoaded handler should be an async IIFE or async function.
